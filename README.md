# LINGORA-Mini-Store


**LINGORA Mini Store** is a front-end website built using **HTML and CSS**. It is designed as a mini online store for students interested in learning languages like **Korean, Mandarin, and other Asian languages**, offering curated learning packs and study kits to support their learning journey.

## Features
- Browse and view different language learning packs
- Clean and organized layout for easy navigation
- Fully responsive design for desktop and mobile screens
- Simple, user-friendly interface

## Problem it Solves
Many students who want to learn foreign languages struggle to find organized study materials in one place. LINGORA Mini Store addresses this by presenting curated learning resources clearly and accessibly, helping learners find materials quickly and efficiently.

## Technologies Used
- HTML
- CSS
- Responsive design principles
